elFinder
========
 - 加入了簡單的帳號與權限功能。
 - 在Toolbar顯示登入ID與登出按鈕。

![DEMO](https://raw.githubusercontent.com/tw1720/elFinder/main/files/%E2%98%85%E5%85%AC%E9%96%8B%E6%96%87%E4%BB%B6%E5%A4%BE(%E5%83%85%E4%BE%9B%E8%AE%80%E5%8F%96)/demo.jpg)

----

設定
-------
 - 在 ./api.php 設定各用戶帳號密碼
 - 在 ./php/connector.minimal.php 設定帳號所屬權限

這個DEMO使用的帳號 / 密碼：
 - admin / admin
 - user1 / user1
 - guest / guest

`詳細的權限設定資訊請參閱官方說明`
